# CreateUUID

ビジネス向けのモダンでセキュアなUUID生成ツールです。Next.jsを使用して構築されています。

## 特徴

- **モダンなデザイン**: InterとOutfitフォントを使用した、清潔感のあるプロフェッショナルなインターフェース。
- **ワンクリック生成**: ボタン一つで5つのUUIDを即座に生成。
- **クリップボード・コピー**: 各UUIDをワンクリックでコピー可能。
- **レスポンシブ**: デスクトップからモバイルまで快適に利用可能。

## 開発環境の実行

以下のコマンドを使用して、ローカル開発サーバーを起動してください。

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認してください。

## 技術スタック

- Next.js 15 (App Router)
- TypeScript
- Vanilla CSS (Glassmorphism design)
- Lucide React (Icons)
